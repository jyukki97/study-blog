## 목표

- Java/JVM 핵심 개념과 컬렉션/동시성 기본기 확보
- 자료구조/알고리즘 패턴 손에 익히기 (정렬, 슬라이딩 윈도우, 그래프 기초)
- OS/네트워크 필수 개념 정리 (프로세스·스레드, HTTP/TCP 흐름)

## 핵심 학습 리스트

- [ ] JVM 메모리 구조, GC 기본, 스레드/락 기초
- [ ] Java 컬렉션 내부 동작 (ArrayList vs LinkedList, HashMap, ConcurrentHashMap)
- [ ] 동시성 기본: synchronized, volatile, Executor, CompletableFuture
- [ ] 자료구조: 배열/해시/스택/큐/힙/그래프
- [ ] 알고리즘 패턴: 정렬, 이진 탐색, 투 포인터/슬라이딩 윈도우, BFS/DFS
- [ ] OS/네트워크: 프로세스/스레드, 컨텍스트 스위칭, HTTP 메서드/상태코드, TCP 3-way

## 실습 과제

- [ ] Custom ThreadPool 간단 구현 (core/max size, queue, graceful shutdown)
- [ ] LRU Cache 구현 (LinkedHashMap extend)
- [ ] 슬라이딩 윈도우/투 포인터 패턴 문제 3개 풀이 후 블로그 작성
- [ ] JVM/GC 키워드로 스터디 노트 1편 작성

## 추천 자료

- Effective Java, Java Concurrency in Practice
- 프로그래머스 고득점 Kit, LeetCode Top Interview 75
- 그림으로 배우는 HTTP & Network, 운영체제 공룡책 요약본
