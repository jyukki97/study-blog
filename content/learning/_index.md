개발하면서 배우고 공부한 내용들을 주제별로 정리합니다. 백엔드 학습은 **기초 → 스프링 → 데이터/설계 → 운영/배포 → Q&A** 모듈로 나누어 지속적으로 업데이트하며, 아래 타임라인·커리큘럼·주제 카드·최신 노트가 자동으로 정렬됩니다.

- **학습 타임라인**: `study_order`를 가진 글이 자동으로 순서대로 표시됩니다.
- **커리큘럼 카드**: Backend Roadmap 토픽 글이 자동으로 순서대로 표시됩니다.
- **주제 카드**: Spring/Database/Kafka/DevOps 등 토픽별 최신 글을 한 번에 확인합니다.
- **최근 학습 노트**: 최신순 정렬, 더보기/토픽별 필터 지원.

## 학습 방법

- **매일 1시간**: 알고리즘/CS를 루틴화
- **주 2회**: 스프링/DB 심화 정리 후 블로그에 기록
- **주말 1회**: 사이드 프로젝트 기능 추가 또는 리팩터링
- **정기 회고**: 2~4주 주기로 회고와 체크리스트 업데이트
